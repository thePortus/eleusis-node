The URL structure is as follows...

To access all entries in a given table, follow the pattern https://eleusis.thePort.us/api/TABLE_NAME/. (e.g. https://eleusis.thePort.us/api/inscriptions/)

To access a single item's detailed entry, follow the pattern https://eleusis.thePort.us/api/TABLE_NAME/ID_NUMBER/ (e.g. https://eleusis.thePort.us/api/inscriptions/300/)

Enjoy and you are free to use the data but please do not redistribute it.
